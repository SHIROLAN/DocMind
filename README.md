# DocMind
 AI-powered Retrieval-Augmented Generation (RAG) system for document question answering using Gemini Embeddings, FAISS, and Streamlit.

## Features

- Multi-PDF Upload
- Semantic Search
- Source Citations
- Chat History
- Gemini Embeddings
- Groq LLM
- FAISS Vector Store
- Streamlit UI

## Tech Stack

- Python
- LangChain
- Groq
- Gemini Embeddings
- FAISS
- Streamlit
