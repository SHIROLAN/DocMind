# DocMind
 AI-powered Retrieval-Augmented Generation (RAG) system for document question answering using Gemini Embeddings, FAISS, and Streamlit.
 
# DocMind

A RAG application built using:

- Python
- LangChain
- Gemini Embeddings
- FAISS
- Streamlit

## Features

- PDF ingestion
- Semantic search
- Vector database retrieval
- Streamlit UI
